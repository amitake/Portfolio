姿勢推定を用いた筋肉トレーニング改善アシストとトレーニング管理システム

python言語でOpenPoseを用いて骨格検出を行い、その骨格情報を基に筋肉トレーニングの反復回数（レップ数）や体の角度を算出し、ユーザのトレーニング内容を可視化を行った. これ筋肉トレーニングの質の向上・怪我や痛みの防止・やる気の継続をアシストする目的である.
